Download

Genet is hosted on Github.

Use git to get the current version:

$ git clone https://github.com/zoggy/genet.git

Then read the Installation section.

Releases
  • 0.2 [2013-03-18].
    • fix handling of explode/implode (foreach construction),
    • add subcommands: genet query fchains, genet query inputs, genet query chains, genet query ichains,
    • genet query filetypes outputs to stdout instead of stderr,
    • add a bash completion script,
    • genet-exec is replaced by subcommand exec of genet,
    • genet-diff is replaced by subcommand diff of genet,
    • genet-query is replaced by subcommand query of genet,
    • genet-chain is replaced by subcommands chain, flatten, and flatten-all of genet.
  • 0.1 [2013-02-21]: Initial release.
