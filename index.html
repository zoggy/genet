Genet

Genet is tool to build a continuous integration platform. It is particulary adapted to developments involving various tools working in chain, that is some tools producing files used by other tools and so on.

This site is under construction, as Genet has not been tested for real yet.

Have a look at screenshots and read the User documentation to have an overview of Genet.

Genet is released under GNU GPL version 3.
