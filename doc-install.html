Installation
Dependencies

Compiling Genet requires that following software to be installed:

  • OCaml >= 4.01,
  • Menhir >= 20122301 (installed with findlib),
  • Xtmpl >= 0.8,
  • Config-file >= 1.1,
  • OCaml-MySQL >= 1.1.1 (should work with prior versions),
  • OCaml-RDF >= 0.8.0,
  • OCamlnet >= 3.6 (with Nethttpd library enabled),
  • Pcre-OCaml >= 6.2.5 (should work with prior versions),
  • Lablgtk >= 2.16.0,
  • Lablgtk-extras >= 1.3,
  • OCamldot >= 1.0,
  • Stog >= 0.10.0 for the documentation.

Genet is also available in OPAM.

Git is required as it is used to identify versions of your project files. Graphviz must also be installed; it is used by the web interface to display clickable graphs. highlight is required to display colored diffs in web interface.

Compiling and installing

After extracting the source tree, go to the extracted directory and type: 

./configureÂ [options]
make all
make install # with sufficient rights

The following files and repositories will be installed, using variables BINDIR, LIBDIR and SHAREDIR set during the configuration step:

  • BINDIR/genet*: the executables.
  • SHAREDIR/genet-web/: stylesheets and images of web interface.
  • SHAREDIR/genet-web/tmpl: templates for web interface.
