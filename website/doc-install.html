Table des matières

Les sources sont gérées sous git sur Gitorious: https://gitorious.org/genet.

Dans le dépôt, l'organisation est la suivante:

  • /utils: utilitaires pour la compilation.
  • /doc: fichiers utilisés pour la génération de documentation. notamment pour OCamldoc mais aussi le vocabulaire RDF utilisé,
  • /website: source du site web, contenant aussi la documentation. Le site est généré avec Stog.
  • /src: les sources des exécutables OCaml et quelques templates.
    • /base: sources de la bibliothèque de base contenant des modules utilisés par les autres parties du logiciel.
    • /indata: sources de la gestion des jeux d'entrée ("inputs") (lecture, création, ...).
    • /grdf: sources de la partie manipulation du graphe sémantique (stockage, modification, extraction, ...).
    • /chains: sources de la gestion des chaines de traitement (parsing, mise à plat, stockage, lecture, recherche, exécution, ...)/
    • /rest: sources de l'interface REST, notamment l'interface web.
    • /driver: sources des programmes principaux, et quelques modules partagés entre ces programmes (options de ligne de commande, ...).
    • /tmpl: templates utilisés pour l'interface web (fichiers XML utilisés avec Xtmpl).
    • /tests: différents fichiers pour les tests.
  • /draft: fichiers de prises de notes, quelques figures, etc.
    • /proval: scripts de tests utilisant les outils de Proval.

La compilation de Genet nécessite l'installation préalable de:

  • OCaml >= 4.00,
  • Menhir >= 20122301 (installed with findlib),
  • Xtmpl >= 0.2,
  • Config-file >= 1.1,
  • OCaml-MySQL >= 1.0.4 (should work with prior versions),
  • OCaml-RDF >= 0.1,
  • OCamlnet >= 3.5.1 (with Nethttpd library enabled),
  • Pcre-OCaml >= 6.2.5 (should work with prior versions),
  • Stog >= 0.2 for the documentation.

Dans le répertoire racine des sources: ./configure [options]

Dans le répertoire racine des sources: make all

Dans le répertoire racine des sources (en ayant des droits suffisants): make install

Les fichiers et répertoires installés sont les suivants, en utilisant les variables BINDIR, LIBDIR et SHAREDIR fixées lors de l'étape de configuration:

  • BINDIR/genet*: les exécutables.
  • SHAREDIR/web/: feuilles de style et images de l'interface web.
  • SHAREDIR/web/tmpl: répertoire des templates de l'interface web.
