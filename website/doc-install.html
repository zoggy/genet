Compiling Genet requires that following software to be installed:

  • OCaml >= 4.00,
  • Menhir >= 20122301 (installed with findlib),
  • Xtmpl >= 0.3,
  • Config-file >= 1.1,
  • OCaml-MySQL >= 1.0.4 (should work with prior versions),
  • OCaml-RDF >= 0.1,
  • OCamlnet >= 3.6 (with Nethttpd library enabled),
  • Pcre-OCaml >= 6.2.5 (should work with prior versions),
  • Stog >= 0.3 for the documentation.

Git is required as it is used to identify versions of your project files. Graphviz must also be installed; it is used by the web interface to display clickable graphs.

./configureÂ [options] make all make install # with sufficient rights

The following files and repositories will be installed, using variables BINDIR, LIBDIR and SHAREDIR set during the configuration step:

  • BINDIR/genet*: the executables.
  • SHAREDIR/web/: stylesheets and images of web interface.
  • SHAREDIR/web/tmpl: templates for web interface.
