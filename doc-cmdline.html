Genet command line tool
Global options
--version print version number and exit
--config <file>use <file> as config file instead of config.txt
-v increase verbosity level
Synopsis tree
  • add add elements to RDF model
    • genet add branch <parent iri> <branch name>
      add new branch
    • genet add diff-command <name> <command>
      add a predefined diff command +
      --force if a diff command with same name exists, replace the associated command
    • genet add filetype <name> <extension> <description>
      add new filetype
    • genet add input <name>
      add new input
    • genet add interface <tool|branch iri> <name>
      add new interface +
      -p <path>path to command (when used, %v will be replaced by version)
      -t <toolname>make interface depend on tool (when used, %{version-name>} will be replaced by version of tool)
    • genet add no-interface <branch|version iri> <interface iri>
      indicate a branch or version does not implement a given interface
    • genet add port <interface iri> <in|out> <filetype-name> [port name]
      add new port +
      -p <n>1-based position to insert port at; default is to append
    • genet add refinst <input name> <chain fullname> <inst chain url>
      set an instanciated chain as reference
    • genet add tool <name>
      add new tool
    • genet add version <(tool|branch) iri> <name>
      add new version
  • chain operations on chains
    • genet chain test <chain file(s)>
      test the given chain files +
      --dot <prefix>generate dot file(s) using <prefix>
    • genet chain tool-combs <flat chain iri>
      show tool combinations usable to instanciate a flat chain
  • genet diff <instanciation1> [<instanciation2>]
    compute differences between executions +
    --diff <name|com>use <com> or command associated to <name> ; default is diff -r -u
    --html output HTML instead of raw diff output
    --keep-files do not erase files after computing diffs
    --url-if-diff if there are diffs, print url instead (useful for nightly reports)
  • genet exec [<input name1> [<input name2> ...]]
    execute inputs +
    --all Execute all inputs
    --force Force execution of already executed chains
  • genet export-rdf
    export RDF graph
  • genet flatten <Mod.chain>
    flatten a chain +
    --dot <prefix>generate dot file(s) using <prefix>
  • genet flatten-all
    flatten all chains
  • genet import-rdf <rdf files>
    import RDF graph
  • genet init-db
    initialize database
  • genet init-dir [<directory>]
    initialize directory +
    --git <repo>will create the 'in' directory by cloning the repository
  • query query information
    • genet query branches [anscestor iris]
      list branches
    • genet query chains
      list chains
    • genet query diff-commands
      list predefined diff commands
    • genet query dot
      output graph in graphviz format
    • genet query fchains
      list flat chains
    • genet query filename <iri>
      print the filename corresponding to the given file iri
    • genet query filetypes
      list filetypes
    • genet query ichains
      list inst chains
    • genet query inputs
      list inputs
    • genet query interfaces [<iri1>] [<iri2> ...]
      list interfaces
    • genet query ref-inst <input name> <chain name>
      print reference inst chain, if any, from given input name and chain fullname
    • genet query ref-inst-of-inst <iri>
      print reference inst chain, if any, from given inst chain
    • genet query tools
      print tools
    • genet query versions [<iri1>] [<iri2> ...]
      list versions
  • remove remove elements from rdf model
    • genet remove port <port iris>
      remove port
  • set set flags
    • genet set active <url of tool version> [true|false]
      set a tool version as active or not
